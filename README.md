**IR Playbook Bot** is a Slack-driven incident response assistant that executes secure PowerShell playbooks from chat commands, delivering enterprise-grade log collection, host quarantine, status checks, and remediation reports with role-based access control (RBAC) and comprehensive audit trails.

---

## ‚ú® Core Capabilities

- **Slack-Native Interface**  
  Execute incident response actions directly from Slack using natural language commands‚Äîno dashboards or custom UIs required.

- **PowerShell Playbook Engine**  
  Production-hardened PowerShell scripts for log collection, host isolation, system status checks, and forensic reporting.

- **Granular RBAC Enforcement**  
  Viewer / Analyst / Responder roles mapped to Slack user IDs with strict zero-trust validation.

- **Immutable Audit Logging**  
  JSON-based command logs capturing actor identity, timestamp, executed action, and execution outcome.

- **Stateful Operation**  
  Disk-backed quarantine tracking and report generation with human-readable Slack delivery.
